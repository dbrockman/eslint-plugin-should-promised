# eslint-plugin-should-promised

Eslint rule for checking that should-promised assertions return


## Install

```
$ npm i -D eslint-plugin-should-promised
```


## Configure

```json
{
  "plugins": [
    "should-promised"
  ],
  "rules": {
    "should-promised/return-promise": 2
  }
}
```
